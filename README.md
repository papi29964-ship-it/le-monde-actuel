# Le Monde-Actuel

Journal en ligne indépendant d'information générale et d'analyse.

## 📰 À propos

Le Monde-Actuel est un site web d'actualités professionnel, moderne et responsive, conçu pour fournir des informations de qualité au public tchadien et international. Le site couvre diverses rubriques d'actualité et offre une expérience utilisateur optimale sur tous les appareils.

## 🚀 Installation et Démarrage

### Prérequis

- **Node.js** (version 18 ou supérieure)
- **npm** ou **yarn**

### Installation

Installer les dépendances :
```bash
npm install
```

### Développement

Lancer le serveur de développement :
```bash
npm run dev
```

Le site sera accessible à l'adresse : `http://localhost:5173`

### Build de Production

Créer une version de production :
```bash
npm run build
```

Les fichiers optimisés seront générés dans le dossier `dist/`

### Prévisualisation de Production

Prévisualiser le build de production :
```bash
npm run preview
```

## 📱 Pages du Site

### Page d'Accueil (/)
- Article principal en vedette
- Grille d'actualités récentes
- Navigation par catégories

### Pages de Catégories (/politique, /economie, etc.)
- Liste des articles par rubrique
- Filtrage par sujet

### Page Article (/article/:id)
- Contenu complet de l'article
- Métadonnées (auteur, date)
- Boutons de partage social
- Articles connexes

### Page Équipe (/equipe)
- Présentation de l'équipe éditoriale
- Directeur de Publication
- Rédactrice en Chef
- Secrétaire de Rédaction
- Journalistes

### Page Contact (/contact)
- Formulaire de contact
- Informations de contact
- Coordonnées téléphoniques
- Réseaux sociaux

## 🎨 Personnalisation

### Couleurs

Les couleurs principales sont définies dans `src/index.css` :

```css
--primary-color: #c41e3a;      /* Rouge principal */
--secondary-color: #1a1a2e;    /* Bleu foncé */
--accent-color: #0f3460;       /* Bleu accent */
```

### Typographie

- **Titres** : Arial, Helvetica, sans-serif
- **Corps de texte** : Georgia, Times New Roman, serif
